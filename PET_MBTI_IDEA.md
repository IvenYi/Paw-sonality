# System Context: Pawsality (宠物 MBTI 测试应用)

## 1. 项目概述 (Project Overview)
- **项目定位**：一款基于 MBTI 框架的宠物性格测试 Web App。
- **核心受众**：18-35 岁中国女性宠物主，重度小红书用户。
- **核心目标**：提供强情绪价值与社交货币（高颜值结果海报），实现高转化率与社交裂变。
- **当前进度**：已完成商业闭环设计、UI/UX 视觉验证、PRD 规划与本土化文案撰写。即将进入 Cursor 本地开发与代码实现阶段。

## 2. 商业模式与用户路径 (Business Model & User Flow)
- **引流与变现**：小红书前端免费引流 -> 网站完成基础测试 -> 付费墙卡点 -> 用户在小红书购买激活码 -> 返回网站核销 -> 解锁高级深度报告。
- **核心漏斗 (12 + 4 法则)**：
  - **免费阶段**：双轨制（猫/狗分开）。完成 12 道极具代入感的行为场景题，获取基础 MBTI 4 位字母（如 ENFP）及简短评价。
  - **付费解锁（The 5th Dimension）**：输入激活码后，解锁附加的 4 道深层情绪题，测出第 5 个字母（如动荡型 -T 或 坚决型 -A）。
  - **终极交付物**：3000 字治愈系专属宠格深度解析 + 专属宠格雷达图 + 宠物与主人的灵魂契合度（需用户输入自己的 MBTI）。
